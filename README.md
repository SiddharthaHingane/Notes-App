# 📝 Notes App

![License](https://img.shields.io/badge/license-Apache%202.0-blue)
![Status](https://img.shields.io/badge/status-active-success)
![Built With](https://img.shields.io/badge/built%20with-HTML%20%7C%20CSS%20%7C%20JavaScript-orange)

A lightweight and responsive **Notes Application** built using **HTML, CSS, and JavaScript**.
The app allows users to create, search, and manage notes efficiently with a clean interface and dark mode support.

---

## ✨ Features

* 📝 **Create Notes** – Quickly write and store notes
* 🔍 **Instant Search** – Find notes in real time
* 🌙 **Dark Mode** – Toggle between light and dark themes
* 💾 **Local Storage** – Notes are saved directly in the browser
* 📱 **Responsive Design** – Works on desktop and mobile devices
* ⚡ **Fast & Lightweight** – No external frameworks required
